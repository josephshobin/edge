DistributableIterableSource

case class DistributableIterableSource[T](iter: Iterable[T], mappers: Int, inFields: Fields = cascading.tuple.Fields.NONE)(implicit set: TupleSetter[T], conv: TupleConverter[T]) extends Source with Mappable[T] with Product with Serializable

This is an implementation of an iterable backed scalding source.

This works in much the same way as does the standard IterableSource but we can distribute the contents across a fixed number of mappers. This is useful for explosive map jobs that use small inputs to produce largs outputs.
